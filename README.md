### Little Tremors, Big Stories

Little Tremors, Big Stories is a Python application that captures drawing and clicking data and uses probabilistic modeling to quantify small variations in hand movement. Users trace shapes, click targets, and then generate session summaries with movement features, charts, and CSV output.

This project is educational and is not intended for diagnosis or medical interpretation.

How to run

Clone the repository:

    git clone <your-repo-link>
    cd little-tremors-big-stories


Install required packages:

    pip install -r requirements.txt


Start the application:

    python ui.py

How to use

    Open the window, draw the shapes, click the targets, and select Analyze to create a timestamped results folder containing graphs, summaries, and CSV data.
